# 🎯 Focus Guard – Chrome Extension

A simple yet powerful Chrome extension designed to help students and learners stay focused while studying online.

---

## 🚀 Features

* 🔒 **Strict Mode**

  * Prevents switching between tabs while enabled
  * Keeps you locked into your study session

* 🚫 **Distraction Blocker**

  * Blocks access to social media websites
  * Stops endless scrolling of reels and short videos

* 📚 **Student-Friendly Design**

  * Built specifically for learners and productivity
  * Lightweight and easy to use

---

## 🧠 Why Focus Guard?

Staying focused while studying online can be difficult due to constant distractions like social media and tab switching.
**Focus Guard** helps you build discipline by enforcing a distraction-free environment.

---

## ⚙️ How It Works

1. Enable **Strict Mode**
2. The extension:

   * Blocks tab switching
   * Restricts access to distracting websites
3. Stay focused until you disable the mode

---

## 🛠️ Installation (Manual)

1. Download or clone this repository
2. Open Chrome and go to:

   ```
   chrome://extensions/
   ```
3. Enable **Developer Mode** (top right corner)
4. Click **Load unpacked**
5. Select the extension folder
